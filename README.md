Created not necessarily purely from "vibe coding" but also not quite "traditional software development" - so I call it AI-powered development.

In this case it's a bot to trade stocks (or options) that meet certain criteria and indicators then get processed for a potential position entrance (buy into the position).

There is a loss prevention where if the position drops 3% below the entrance price then the position is auto liquidated. Similarly, if the stock hits +10% then it triggers an auto-sell.

Discord notifications is easier (and free) than many other options while still being informed (near real-time) about the entry into a new stock / option (or the selling of a position).
